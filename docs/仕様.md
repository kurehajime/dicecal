# 仕様

サイコロカレンダー

## 技術スタック
* React
* TypeScript
* Vite
* Three.js
* React Three Fiber
* Rapier

## 画面構成

* サイコロを4つ斜めに配置できる台座
* サイコロ4つ
  * 月を表すサイコロ
    * JUN/DEC
    * FEB/AUG
    * MAR/SEP
    * APR/OCT
    * MAY/NOV
    * JAN/JUL
  * 曜日を表すサイコロ
    * MON
    * TUE
    * WED
    * THU
    * FRI
    * SAT/SUN
  * 日を表すサイコロ1
    * 0
    * 1
    * 2
    * 6
    * 7
    * 8
  * 日を表すサイコロ2
    * 0
    * 1
    * 2
    * 3
    * 4
    * 5

## 操作手順
任意のサイコロをクリックする。
クリックするとサイコロが持ち上がり、画面真正面を向く。

選択されたサイコロの周りには以下のUIが表示される。
* サイコロの上下左右の方向に回転するための矢印が表示される。
* 加えて左右に回転するための矢印も表示される。
* 決定のためのチェックマークが表示される。

決定をすると元の位置に戻り、サイコロの状態が保存される。
